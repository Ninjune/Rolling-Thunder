\doxysubsection{Detailed Description}
This class stores the data needed for entities\textquotesingle{} animations. Split into sections\+: named with what the purpose is. 

Sections include\+:
\begin{DoxyItemize}
\item name
\item the texture\texorpdfstring{$\ast$}{*}
\item horizontal\+: how many frames on the x axis
\item vertical\+: how many frames on the y axis
\item vec2i start (top left)
\item vec2i end (bottom right)
\end{DoxyItemize}

Based on this info, it will automatically split the section into frames. \doxylink{class_animation_data}{Animation\+Data} is a collection of sections for specific entities.

Why are we doing this? Many of the frames in the spritesheets are spaced differently. Each section has consistent spacing. It also allows us to split them up by their action, such as "{}\+Move\+Left"{} or "{}\+Die"{}

Note\+: Frames go from top-\/left to bottom-\/right or left to right if vertical\+Frames = 1
