\doxysubsection{Detailed Description}
This class controls the \doxylink{class_g_u_i}{GUI} of the game. 

For the main menu\+: make a recording until the player presses 5, where we will switch to ingame animation. For the leaderboard, we can edit the video and replace the included leaderboard with the blue color, then add in our own during those certain frames.
