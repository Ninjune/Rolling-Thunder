# Rolling Thunder
Recreation of the Rolling Thunder arcade game from 1986.
## Documentation
documentation can be found doc\html/index.html
## controls
walk left-> D
walk right-> A
look  up -> W
jump -> F
shoot -> R
climb Up -> W+F
climb down -> S+F
